### 切削の流れ  

3D データを作る >> Fusion360 の CAM で切削パスを生成 >> パスを chilipeppr に回して CNC を動かす  


---  


### 環境  

CNC : r7 cnc  
CNC の制御 : chilipeppr （[http://chilipeppr.com/](http://chilipeppr.com/)）  
CAM : Autodesk Fusion360  

ぐぐったら、公式リファレンスがあった。これをちゃんと見るのが良さそう。
